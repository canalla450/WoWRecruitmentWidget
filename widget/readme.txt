World of Warcraft Recruitment - HereticGaming.com

== Description ==

This plugin adds a widget allowing you to display recruitment information for each class.

Allows you to select a recruitment priority from:

* High
* Medium
* Low
* Closed

== Installation ==

1. Use WordPress' built-in installer
